### Installation

1. Get a free API Key at [https://developers.google.com/books](https://developers.google.com/books)
2. Clone the repo
   ```sh
   git clone https://github.com/yuanwuu/react-bookshelf.git
   ```
3. Install NPM packages
   ```sh
   npm install
   ```
4. Install Bootstrap packages
    ```sh
    npm install react-bootstrap bootstrap
    ```
5. Enter your API in `/src/pages/Home.js`
   ```js
   const apiKEY = 'ENTER YOUR API';
   ```
